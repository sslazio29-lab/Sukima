# AGENTS.md（Codex / Claude Code 共通の入口）

このプロジェクト **Sukima** で作業する全エージェントへ。

## 読むものと頻度（コンテキスト節約のため階層化）

- **毎回・必ず**：`CLAUDE.md` … 改変不可の不変ルール。最優先。ここだけは毎回読む。
- **作業を再開するとき**：`WORKLOG.md` … 直近の作業状況。続きから着手する。
- **機能を実装するとき**：`SPEC.md` の**該当箇所**を参照（全文を毎回読む必要はない）。
- **参照のみ**：`Sukima_引き継ぎ書_v2.md` … プロジェクトの背景・思想・経緯の完全記録。
  **初回オンボーディング時、または「なぜそう決めたか」を確認したいときだけ**読む。
  実務で効く歯止めは `CLAUDE.md` に抽出済みなので、毎回フルで読む必要はない。

`CLAUDE.md` と `AGENTS.md` の内容が食い違った場合は **`CLAUDE.md` を優先**すること。

## このプロジェクトの一言

Claude が作った3択問題を、受験生（依頼者の子・京大/阪大志望）が iPhone で
サクサク解く、全科目対応の暗記特化アプリ。**「パッと始めてパッと終われる」が最優先。**

## 絶対の歯止め（詳細は CLAUDE.md）

- やることは **3択を解く**、ただ1つ。やり込ませる機能は作らない。
- **GitHub Pages の静的アプリで完結**。サーバー処理・Cloudflare を持ち込まない。
- 迷ったら「足す」より「足さない」。シンプルさが最大の防御。

## 作業の進め方

- 設計・仕様の変更は **依頼者の承認を得てから**着手する。
- 小タスクごとに **コミット・プッシュ**。大タスク終了時に各 md の肥大化を点検・最適化。
- 作業内容は毎回 `WORKLOG.md` に記録する。
